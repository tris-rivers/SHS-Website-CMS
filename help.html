Help

  1. Home

    1.1 The video that is displayed in the homepage of the website can be changed here by pasting the Youtube link of the desired video in the textfield.
    1.2 Upload the video privately on YouTube then specify the link in the input field provided.

  2. About

    2.1 Here in the about page you can edit the Vision, Mission, Identity, Graduate Attributes and History that will be displayed in the website.
    2.2 Each tab under the about page includes a text editor where you can edit the text in the About page of the website. [insert visual editor tutorial if needed]

  3. Faculty

    3.1 Under the faculty page are the Administration, Strand Chairs and Faculty Members tabs where you can edit the list of Faculty members that are under the SHS administration.
    3.2 Each tab includes a table where you can add and edit the list of faculty members.

  4. News

    4.1 Here in the news page you can add, edit and delete announcements that are shown in the website.
    4.2 Under the news page includes the Announcements, Events and Gallery pages.
    4.3 In the announcements and event pages there is a table where the they are listed. Each entry includes the date published, title, description and image that will be featured in the website.
    4.4 In the gallery page you can add and delete images that are displayed in the website.

  5. Strands

    5.1 Here you can view and edit the list of the available strands in the UST SHS including the description and image of each strand.
    5.2 In the Researches tab you can add, edit and delete sample research papers from the different strands.

  6. Downloadables

    6.1 Here you can add, edit and delete the different files that the users may use.
    6.2 The files that can be given to the users may be attached to its title.
    6.3 It is advisable to upload a pdf file instead of a doc file to avoid human error when editing the said file.

  7. Admission

    7.1 Here you can edit the page where the users will be redirected to when they select the "Admission" page.

  8. Chatbot

    8.1 Here in the chatbot page is a (list) of the inquiries and answers for those inquiries.
    8.2 You may edit the inquiries with the use of the (text editor).
